# Planty

Planty is a Streamlit RAG chatbot about Plants, Herbs, Flowers, Crops, and trees.

## Features
- Answers questions based on a custom plant knowledge base
- Uses retrieval-augmeted generation (RAG)
- Build with Streamlit, LlamaIndex, and Groq

## Run locally
```bash
streamlit run app.py
```

Example questions:
- What is Rosemary used for?
- What family does Bluebell belong to?
- Why is Aloe Vera well known?
